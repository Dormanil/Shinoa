**Administration**
Note: Restricted to the bot owner.
`[PREFIX]setavatar <image url>` - Sets the bot's avatar to the given image. Image will be stretched to fit if it isn't square.
`[PREFIX]stats` - Shows various information about the bot instance.

**Moderation**
Note: Muting requires your server to have a `Muted` role that is given to muted users.
`[PREFIX]mute <@user> [x seconds/minutes/hours]` - Mutes the given user, with an optional timespan. Requires the `Mute Members` permission.
`[PREFIX]unmute <@user>` - Unmutes the given user. Requires the `Mute Members` permission.
`[PREFIX]ban <@user>` - Bans the given user. Requires the `Ban Members` permission.
`[PREFIX]kick <@user>` - Kicks the given user. Requires the `Kick Members` permission.

**Greetings**
Note: Requires the `Manage Server` permission.
`[PREFIX]greetings enable` - Enable greeting messages for the server, and bind them to the current channel. 
`[PREFIX]greetings disable` - Disable greeting messages for the server. 
`[PREFIX]greetings here` - If enabled, move the server's greeting messages to the current channel. 

**Anime/Manga**
`[PREFIX]anime <query>` - Search MyAnimeList for the given anime, and show the first result.
`[PREFIX]manga <query>` - Search MyAnimeList for the given manga/LN, and show the first result.

**Sword Art Online Wikia**
`[PREFIX]saowiki <query>` - Searches the SAO Wikia for the given query and shows the first result.

**Japanese Dictionary**
`[PREFIX]jp <query>` - Searches jisho.org for the given query and shows the first result.

**Wikipedia**
`[PREFIX]wiki <query>` - Searches Wikipedia for the given query and shows the first result.

**Anime Notifications**
Note: All commands also work when PMing the bot.
`[PREFIX]animefeed enable` - Enable anime notifications for the current channel.
`[PREFIX]animefeed disable` - Disable anime notifications for the current channel.

**Reddit**
Note: All commands also work when PMing the bot.
`[PREFIX]reddit add <subreddit>` - Bind notifications for a subreddit to the current channel.
`[PREFIX]reddit remove <subreddit>` - Remove notifications for a subreddit from the current channel.
`[PREFIX]reddit list` - List all subreddits bound to the current channels.

**Twitter**
Note: All commands also work when PMing the bot.
`[PREFIX]twitter add <username>` - Bind notifications for a user to the current channel.
`[PREFIX]twitter remove <username>` - Remove notifications for a user from the current channel.
`[PREFIX]twitter list` - List all users bound to the current channels.
